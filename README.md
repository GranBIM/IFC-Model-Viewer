# IFC-Model-Viewer
Komplex IFC viser med mange funksjoner, noen av de er:
Navigering, kontroll av data, sette mål, isolere objekter.
Støtter også: Strømming av modeller, sammenslåing av mesh, rendring kun av synlige objekter, justering av oppløsning (Dette er for ytelse)

Støtter: Kjøretøy med fysikk, kjør rundt i IFC modellen med tastatur, Xbox kontroller, 3DConnecion, etc...
3D person, 1st person, med justeringsmuligheter..
Analyse: Slipp en ball på overflaten å se hvor den triller, friksjon, gravitasjon, etc kan justeres
Analyse: Slipp dråper i en grid for å raskt se hvor vann samler seg (Under utvikling) Ytelse begrenset som nettbasert løsning

Alt er under utvikling...
