# IFC-Model-Viewer
Dette er en avansert IFC modellviser med 100% støtte for IFC 2x3 / 4 / 4x3
utviklet for å forenkle bruk og forståelse av modell.

Med bakgrunn som BIM tekniker har jeg utviklet dette verktøyet som skal gjøre utveksling, kommunikasjon og forståelse lettere for arkitekter og privatpersoner.
#Hovedfunksjoner:
Kan laste å vise ubegrenset antall modeller samtidig, dette innebærer også synlighet per modell.
Støtte for å lese modelltre og Pset på objekt, støtter også isolering av enkeltobjekter eller samlede.
Kamera har standardvisninger og støtte for Perspektiv / Ortografisk visning "rettvinklet".
Modellvisning med støtte for: Solid, trådramme og røntgen. Røntgenvisning er også objektspesifik ved bruk av verktøy "Fall" Objekter i kontakt med verktøy forblir synlig mens resten får røntfenvisning, ved aktivering.
Modellvisning støtter også "tilbakestilling" og autorotering i begge retninger med hastighetsjusteringer.
Måleverktøy: Mål høyde / posisjon, lengde eller fall. for fall gir modus "normal" utslag ved helning brattere enn 1:15, med TEK17 aktivering er det tilpasset basert på lengde.
Simulering: Simuler vann over terreng med permanent stripe på overflate for å enkelt se hvor vannet renner.
Kjøretøy: Kjør biler i ulike størrelser eller simuler en tur med buss ved å først tegne rute med nurbs verktøy direkte på overflaten. 
Lys: Tilpass lyset i modell, farge og styrke, tilsvarende for kamera når det nærmer seg modellen.
Gamemode: Endre kamera i kjøretøy, tilpass kontrollene for spacemuse eller Xbox kontroller.
Utviklerverktøy: Avanserte funksjoner for rendring av modell og objekter, oppløsning og liste over IFC entiteter, disse kan deaktiveres ved ønske for å øke ytelsen.
Tema: Bytt mellom lys og mørk modus.
